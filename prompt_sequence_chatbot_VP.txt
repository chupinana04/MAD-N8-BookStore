Draw a Sequence Diagram for the Chatbot (Virtual Assistant) feature of a BookStore Android app using RAG (Retrieval-Augmented Generation) architecture.

Objects:
Actor "User"
Boundary "ChatbotScreen"
Control "BookStoreApiService"
Control "Backend"
Control "Groq_API"
Entity "Database"

Messages:

1. User to ChatbotScreen: Input message and click Send
2. ChatbotScreen to ChatbotScreen: Add user message to UI, set isLoading=true
3. ChatbotScreen to BookStoreApiService: chatbot(ChatbotRequest)
4. BookStoreApiService to Backend: POST /api/ai/chatbot
5. Backend to Groq_API: Extract keyword from message (Llama-3.1)
6. Groq_API to Backend: return keyword
7. Backend to Database: Query books by keyword (Author or Title)
8. Database to Backend: return context books
9. Backend to Backend: Pre-process book data (format price, physical specs)
10. Backend to Groq_API: Send prompt with context books & user message (Llama-3.3)
11. Groq_API to Backend: return AI reply
12. Backend to BookStoreApiService: return ChatbotResponse(reply)
13. BookStoreApiService to ChatbotScreen: return Response<ChatbotResponse>
14. ChatbotScreen to ChatbotScreen: Add AI reply to UI, set isLoading=false
15. ChatbotScreen to User: Display AI reply
